Bittrex API Key Instructions

Go to Bittrex.com

You can get your keys from Bittrex by going to the API Keys option of the Settings drop-down menu.

Use Two-Factor Authentication

You must have Two-Factor Authentication enabled in order to be able to create keys. Most people simply use the Android or iOS version of Google Authenticator for this.

Create Read-Only Keys

It is always best-practice to only give the access level you need to an external application.
